# Incident Response & Digital Forensics Investigation

## Overview

This repository documents a simulated incident response and digital forensics investigation involving a targeted spear-phishing attack against a hybrid cloud organization. The project follows the National Institute of Standards and Technology (NIST) SP 800-61 Incident Response Lifecycle and demonstrates how an incident response team detects, analyzes, contains, eradicates, and recovers from a cybersecurity incident while preserving digital evidence.

This project was created to demonstrate incident response planning, forensic documentation, evidence handling, and technical reporting using industry-recognized best practices.

---

## Scenario

A mid-sized technology company experienced a targeted spear-phishing attack that resulted in unauthorized access to employee email accounts. After gaining access, the attackers deployed malware and attempted to exfiltrate sensitive business information to an external server.

The incident response team investigated the attack, identified the attack vector, preserved digital evidence, removed the malware, restored affected systems, and documented recommendations to reduce the likelihood of future incidents.

---

## Objectives

The objectives of this project were to:

- Investigate a simulated cybersecurity incident
- Follow the NIST SP 800-61 Incident Response Lifecycle
- Identify the attack vector
- Preserve digital evidence
- Document incident response procedures
- Produce professional forensic documentation
- Recommend security improvements
